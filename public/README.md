# public/

Place your static assets here — profile photo, OG image, project screenshots, favicons, etc.

This directory is intentionally empty in the template. The live site's personal assets are not included.

## Suggested structure

```
public/
  profile.jpg           ← your profile photo (used in hero)
  og-image.png          ← open graph image (1200 × 630)
  favicon.ico
  logo.png
  experience/           ← company logos
  projects/             ← project screenshots & demo videos
```
